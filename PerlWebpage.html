Perl Programming Language

Perl was invented by Larry Wall in 1967, it was used as a text processing language.

Perl is supposedly an easy language to learn. It looks like most programming languages you're likely to have experience with, so if you've ever written an C program, an awk script, a shell script, or even BASIC program, you're already part way there.

Perl is an interpreted language. This means that you can write your programs and test them without an intermediate compilation step, allowing you to experiment and test/debug quickly and easily. This ease of experimentation flattens the learning curve even more.

This is the Perl homepage. You can download Perl, learn how to get started, view the documentation, and other things on their site.
